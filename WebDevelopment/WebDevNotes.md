## CSS
Types of invoking CSS files:  
* Inline
* Internal
* External  
  
Types of CSS selectors:  
* Element Selector: h1{ color: red; }  
* Class Selector: <h1 class="new_header"> ==> .new_header{ color: yellow; }  
* Attribute Selector: <h1 id="new_id"> ==> #new_id{ color: green; }
